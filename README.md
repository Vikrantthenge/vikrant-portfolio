Locally deployed Generative AI assistant using Ollama, Docker, and Open WebUI to explore prompt-engineered workflows for operational risk analysis, KPI assessment, recurring-pattern identification, and executive-style reporting.

The project demonstrated how a self-hosted language model could assist with reviewing operational information and producing structured management summaries without sending data to an external cloud service.
